# Dino Jump Game

A faithful recreation of the classic Chrome Dino game, featuring authentic mechanics and sprites.

## Features
- **Authentic Sprites**: Extracted and mapped using **Google Gemini AI Pro**.
- **Advanced Mechanics**: Variable jump height, ducking, and speed progression.
- **Obstacles**: Various cactus groups and flying birds.

## How to Run

1.  Ensure you have Python installed.
2.  Install Pygame:
    ```bash
    pip install pygame
    ```
3.  Run the game:
    ```bash
    python dino_game.py
    ```

## Controls
- **SPACE** or **UP**: Jump (Hold for higher jump)
- **DOWN**: Duck (Avoid mid-level birds)
