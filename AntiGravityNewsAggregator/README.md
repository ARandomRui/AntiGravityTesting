# AntiGravity News Aggregator

A simple news aggregator application that fetches headlines from the NewsAPI and displays them in a clean, readable format.

## Features
- Fetches top headlines from NewsAPI.
- Displays news articles with titles, descriptions, and images.
- Simple Node.js/Express backend.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Environment Setup**:
    - Ensure you have a `.env` file with your `NEWS_API_KEY`.
3.  **Start the Server**:
    ```bash
    npm start
    ```
    or for development:
    ```bash
    npm run dev
    ```
4.  Open your browser and navigate to `http://localhost:3000` (or the port specified in your console).
