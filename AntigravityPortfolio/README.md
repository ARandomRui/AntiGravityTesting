# Antigravity Portfolio

A personal portfolio website showcasing projects and providing a contact form for visitors.

## Features
-   **About Section**: Introduction to the user.
-   **Projects Showcase**: Display of recent work.
-   **Contact Form**: Simple form for inquiries.
-   **Responsive Design**: Works on mobile and desktop.

## How to Run

1.  Open the project folder.
2.  Double-click `index.html` to open it in your web browser.
3.  Alternatively, use a local server:
    ```bash
    python -m http.server
    ```
    Then visit `http://localhost:8000`.
4. Developer Notes: For some reason Antigravity can't get the point that the above doesn't work. Use No 5. 
5.  Alternatively, use a local server:
    ```bash
    python -m http.server --bind 127.0.0.1 8000
    ```
    Then visit `http://127.0.0.1:8000/`.
